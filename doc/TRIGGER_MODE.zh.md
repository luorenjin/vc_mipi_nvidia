# 触发模式

通过以下命令激活触发模式：
```
v4l2-ctl -c trigger_mode=<trigger mode number>
```
触发模式将保持设置状态，直到使用以下命令停用：
```
v4l2-ctl -c trigger_mode=0
```
以下是用于说明每种模式具体行为的时序图。

## 外部和脉宽触发模式 (1 或 2)
![外部触发模式](../doc/plantuml/tm_external.svg)

## 自触发模式 (3)
此功能使用再触发周期寄存器（reg13-16）来模拟自触发模式，该模式看起来像原生流模式，但具有纳秒级精确的快门速度设置和闪光灯触发输出。在此模式下，外部触发硬件信号被禁用。 

请注意，在此模式下，最大帧率取决于传感器的读取时间加上快门时间（相比之下，在流模式下，传感器可以在读取的同时进行曝光）。

![自触发模式](../doc/plantuml/tm_self.svg)

## 单次触发模式 (4)
通过执行以下命令触发图像采集：
```
v4l2-ctl -c single_trigger=1
```
![单次触发模式](../doc/plantuml/tm_single.svg)

## 自触发和同步触发模式 (3 和 5)
此模式用于通过主/从同步方式同步两个或多个传感器模块。

请注意，您必须启用主传感器的闪光灯输出。将主传感器的闪光灯输出连接到从传感器模块的触发输入。

![自触发和同步触发模式](../doc/plantuml/tm_masterslave.svg)

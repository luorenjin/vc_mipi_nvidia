# IO 模式

通过以下方式激活 IO 模式：
```
v4l2-ctl -c io_mode=<IO 模式编号>
```
IO 模式将保持设置，直到通过以下方式停用：
```
v4l2-ctl -c io_mode=0
```
下面是说明每种模式具体行为的时序图。

在所有模式下，您都可以按照 [触发模式](TRIGGER_MODE.md) 中的说明激活触发模式。 

## 禁用 (0)
在此模式下，闪光灯信号被禁用，触发信号为高电平有效。

![禁用](../doc/plantuml/iom_disabled.svg)

## 闪光灯高电平有效 (1)
在此模式下，闪光灯信号为高电平有效，触发信号为高电平有效。

![闪光灯高电平有效](../doc/plantuml/iom_flash_active_high.svg)

## 闪光灯低电平有效 (2)
在此模式下，闪光灯信号为低电平有效，触发信号为高电平有效。

![闪光灯低电平有效](../doc/plantuml/iom_flash_active_low.svg)

## 触发低电平有效 (3)
在此模式下，闪光灯信号被禁用，触发信号为低电平有效。

![触发低电平有效](../doc/plantuml/iom_trigger_active_low.svg)

## 触发低电平有效且闪光灯高电平有效 (4)
在此模式下，闪光灯信号为高电平有效，触发信号为低电平有效。

![触发低电平有效且闪光灯高电平有效](../doc/plantuml/iom_trigger_low_flash_high.svg)

## 触发和闪光灯均为低电平有效 (5)
在此模式下，闪光灯信号为低电平有效，触发信号为低电平有效。

![触发和闪光灯均为低电平有效](../doc/plantuml/iom_trigger_flash_active_low.svg)
